Project: House Price Prediction using Linear Regression

Description:
This project predicts house prices based on features such as area, bedrooms, and bathrooms using a Linear Regression model.

Steps:

1. Data preprocessing
2. Model training
3. Prediction
4. Evaluation using MSE and R2 score

How to Run:

1. Open Jupyter Notebook
2. Run all cells

Libraries Used:

* pandas
* numpy
* sklearn
* matplotlib
* seaborn
