Project: House Price Prediction using Linear Regression

Description:
This project predicts house prices using Linear Regression and Regularization techniques.
The goal is to build a machine learning model that can estimate house prices based on different features such as area, location, building type, and other property characteristics.

Steps:

1. Perform data preprocessing
2. Encode categorical features
3. Split the dataset into training and testing sets
4. Apply feature scaling
5. Train a Linear Regression model
6. Improve the model using L1 (Lasso) and L2 (Ridge) regularization
7. Perform hypothesis testing using p-values
8. Evaluate model performance using multiple metrics
9. Visualize predictions and errors

How to Run:

1. Open Jupyter Notebook
2. Run all cells

Libraries Used:

* pandas
* numpy
* sklearn
* matplotlib
* seaborn
*Statsmodels
